如果你有自己的 AI Agent 应用，不管它基于 AI SDK、LangGraph、Pi、Claude SDK、OpenAI SDK 还是自研 HTTP 接口，都可以用 `defineAgent` 接到 fasteval。

这类 eval 通常走 **直连模式**：不需要 Docker，不需要给 agent 一个文件系统，只需要 adapter 调用你的服务，并把响应映射成 fasteval 标准事件流。

## 适合评什么

- 对话回复是否准确、礼貌、稳定。
- 工具调用是否发生，参数是否正确。
- 结构化输出是否符合 schema。
- 检索、记忆、订单、客服等业务流程是否跑通。
- 不同模型、prompt、检索配置和 feature flag 的效果。
- 成本、token 和延迟是否在预算内。

## 写 adapter

```ts agents/support-api.ts
import { defineAgent } from "fasteval";

export default defineAgent({
  name: "support-api",
  capabilities: {
    conversation: true,
    toolObservability: true,
  },
  async send(input, ctx) {
    const res = await fetch(process.env.SUPPORT_API_URL!, {
      method: "POST",
      headers: { "content-type": "application/json" },
      body: JSON.stringify({ message: input.text }),
      signal: ctx.signal,
    });

    const body = await res.json();
    return {
      status: "completed",
      data: body,
      events: [
        { type: "message", role: "assistant", text: body.reply },
        ...toToolEvents(body.toolCalls ?? []),
      ],
      usage: body.usage,
    };
  },
});

function toToolEvents(toolCalls: Array<{ id: string; name: string; input: unknown }>) {
  return toolCalls.map((call) => ({
    type: "action.called" as const,
    callId: call.id,
    name: call.name,
    input: call.input,
  }));
}
```

Adapter 的核心职责是两件事：调用你的 agent，并把返回值映射成 `StreamEvent[]`。eval 文件不应该理解你的服务私有响应格式。

## 注册 agent

```ts fasteval.config.ts
import { defineConfig } from "fasteval";
import supportApi from "./agents/support-api";

export default defineConfig({
  agents: [supportApi],
  defaultAgent: "support-api",
  maxConcurrency: 4,
  timeoutMs: 300_000,
});
```

## 写 eval

```ts evals/support/refund.eval.ts
import { defineEval } from "fasteval";
import { includes, equals } from "fasteval/expect";

export default defineEval({
  description: "退款意图识别并查询订单",
  async test(t) {
    const turn = await t.send("I want a refund for order A123.");

    t.succeeded();
    t.calledTool("lookup_order", { count: 1 });
    t.check(turn.data, equals({ intent: "refund", orderId: "A123" }));
    t.check(t.reply, includes("refund"));
    t.maxCost(0.05);
  },
});
```

## 本地和线上都能跑

```bash
SUPPORT_API_URL=http://localhost:3000/api/agent npx fasteval support
SUPPORT_API_URL=https://api.example.com/agent npx fasteval support
```

URL 是 adapter 的配置，不是 CLI 位置参数。位置参数只过滤 eval ID。

## 做实验

把不同模型、prompt 或检索配置放进 experiment：

```bash
npx fasteval exp support-regression
```

重点看：

- pass rate。
- judge 分数和失败原因。
- 平均 latency。
- token 和 cost。
- 工具调用参数是否稳定。
