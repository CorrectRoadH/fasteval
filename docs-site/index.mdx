fasteval is a lightweight TypeScript library for evaluating AI agents. Whether you're testing a conversational assistant, a coding agent like Claude Code or Codex, or an in-process function, fasteval gives you a single `defineEval` API — with built-in scoring, concurrency scheduling, and structured result artifacts.

## Why "fast"

<CardGroup cols={3}>
  <Card title="Fast to write" icon="feather">
    One file = one eval. ID auto-derived from path. No boilerplate, no callback hell.

## Get started

<Steps>
  <Step title="Install fasteval">
    ```bash
    npm install -D fasteval
    ```
  </Step>
  <Step title="Scaffold your project">
    ```bash
    npx fasteval init
    ```
    This generates `evals/`, `fasteval.config.ts`, and an example eval file.
  </Step>
  <Step title="Write your first eval">
    ```typescript evals/weather/brooklyn.eval.ts
    import { defineEval } from "fasteval";
    import { includes } from "fasteval/expect";

    export default defineEval({
      description: "Brooklyn weather query",
      async test(t) {
        await t.send("What's the weather in Brooklyn today?");
        t.succeeded();
        t.calledTool("get_weather", { input: { city: "Brooklyn" } });
        t.check(t.reply, includes("sunny"));
      },
    });
    ```
  </Step>
  <Step title="Run it">
    ```bash
    npx fasteval weather
    ```
  </Step>
</Steps>
